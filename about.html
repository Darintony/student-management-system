EduTech Pro

Student Registration & Management System

A modern, multi-page student management platform that streamlines enrollment, record-keeping, and academic analytics — built entirely with vanilla HTML5, CSS3, and JavaScript ES6. No frameworks. No backend. Just the web platform used well.

🎯

Project Name & Purpose

EduTech Pro helps educational institutions manage student registrations efficiently. It provides a complete CRUD interface to register, track, search, filter, and maintain student records — all persisted in the browser with no server needed.

  • Register students with photo upload
  • Auto-generated unique Student IDs (STU-XXXX)
  • Real-time search and multi-filter
  • Analytics dashboard with recent data & charts
  • Full offline capability via LocalStorage
⚙️

Technologies Used

Built with standard web technologies — zero dependencies, no build tools:

HTML5 CSS3 JavaScript ES6+ Local Storage API CSS Grid CSS Flexbox CSS Custom Properties FileReader API Base64 Encoding Plus Jakarta Sans CSS Animations Responsive Design
✨

Key Features

  • Full CRUD — Create, Read, Update, Delete students
  • Photo upload with drag & drop, live preview & Base64 storage
  • Auto-generated unique STU-XXXX Student IDs with regenerate
  • Inline form validation — 8 rules including uniqueness checks
  • Live address character counter (warns at 170+)
  • Simultaneous search + gender + course filter
  • Student detail view modal with all fields
  • Confirm-before-delete modal
  • Top-left slide-in toast notifications (4 types)
  • Fully responsive — mobile hamburger, tablet, desktop
  • XSS-safe HTML output escaping throughout
  • Accessible — ARIA roles, labels, live regions
